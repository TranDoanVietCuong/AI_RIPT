# Machine Learning Roadmap: Supervised vs Unsupervised Learning

## 1. Lý thuyết Nền tảng
### 1.1. Supervised Learning (Học có giám sát)
- Định nghĩa và Bản chất
- Quy trình xây dựng mô hình (7 bước tiêu chuẩn)
- Phân biệt Classification (Phân loại) và Regression (Hồi quy)

### 1.2. Unsupervised Learning (Học không giám sát)
- Định nghĩa và Bản chất
- Mục tiêu chính & Ứng dụng thực tế
- Phân biệt Clustering (Gom nhóm) và Dimensionality Reduction (Giảm chiều)

## 2. Tổng quan So sánh & Thuật toán
### 2.1. Bảng so sánh Supervised vs. Unsupervised
- Tiêu chí: Dữ liệu đầu vào, Mục tiêu, Độ phức tạp, Cách đánh giá

### 2.2. Các thuật toán tiêu biểu
- **Supervised:** Linear Regression, Logistic Regression, Decision Tree, Random Forest, SVM, KNN
- **Unsupervised:** K-Means, DBSCAN, Hierarchical Clustering, PCA, t-SNE

## 3. Lộ trình Thực hành & Dataset
### 3.1. Nguồn Dữ liệu (Datasets)
- Iris Dataset (Classification)
- Wine Dataset (Classification & Dim Reduction)
- Mall Customer Segmentation (Clustering)

### 3.2. Nội dung Bài tập
- **Phần A: Supervised Learning**
    - Tiền xử lý dữ liệu
    - Huấn luyện mô hình đa dạng
    - Đánh giá hiệu suất (Accuracy, Precision, Recall, F1)
- **Phần B: Unsupervised Learning**
    - Phân cụm khách hàng (K-Means, DBSCAN)
    - Giảm chiều dữ liệu & Trực quan hóa (PCA, t-SNE)
