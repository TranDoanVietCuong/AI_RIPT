# Tổng quan nội dung lý thuyết tuần 1
## 1. Tổng quan về AI
* Khái niệm & Lĩnh vực: AI, Machine Learning, Deep Learning, NLP, Computer Vision.
* Dữ liệu: Vai trò và tầm quan trọng.
* 3 Loại bài toán ML: Supervised (Có giám sát), Unsupervised (Không giám sát), Reinforcement (Tăng cường).
* Thuật ngữ & Ký hiệu: Dataset (Train/Test/Val), Features ($x$), Label ($y$), Model, Loss Function, Overfitting.
## 2. Các thư viện Python cốt lõi
* Numpy: Tính toán mảng, ma trận.
* Pandas: Thao tác dữ liệu dạng bảng (DataFrame).
* Matplotlib: Trực quan hóa dữ liệu (Vẽ biểu đồ 2D).
* PyTorch: Xây dựng mô hình Deep Learning.
## 3. Môi trường & Nguồn dữ liệu
* Công cụ (IDE): Jupyter Notebook, Google Colab.
* Nguồn tài nguyên:
  * Kaggle: Kho dữ liệu (Datasets) và các cuộc thi AI.
  * Hugging Face: Kho mô hình (Models) và dữ liệu chuyên về NLP/Audio/Vision.
## 4. Quy trình EDA (Phân tích dữ liệu khám phá)
* Data Understanding: Kiểm tra cấu trúc, kiểu dữ liệu, lỗi/missing values.
* Data Analysis: Thống kê mô tả (Mean, Median, Std...).
* Data Visualization: Biểu diễn dữ liệu bằng hình ảnh (Histogram, Boxplot, Heatmap...).
